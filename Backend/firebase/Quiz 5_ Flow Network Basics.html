Match each of the following with the most suitable phrase.  Phrases may be reused

Σ xj,n for all j
edge capacity

Σ x1,j for all j

source vertex

Σ xj,i  =  Σ xi,j  where the sums are over j 

sink vertex

0 ≤ xi,j ≤ ui,j  for all i and j

Feedback

The correct answer is: Σ xj,n for all j → value of a flow, edge capacity → denoted ui,j, Σ x1,j for all j → value of a flow, source vertex → The vertex in a flow network with no incoming edges, Σ xj,i  =  Σ xi,j  where the sums are over j  → flow conservation constraint, sink vertex → The vertex in a flow network with no outgoing edges, 0 ≤ xi,j ≤ ui,j  for all i and j → capacity constraints

Question text

Match each of the following with the most suitable phrase.  Phrases may be reused.

Assume the following notation:

G is a flow network with vertices V and edges E, and integer capacities (weights) ui,j > 0 for each edge (i,j) in E.

If there is a flow on G then xi,j represents the flow on the edge (i,j) in E.

An augmenting path is found using ____________ on the residual graph

forward edge

 ui,j - xi,j  

xj,i
backward edge

residual graph

Feedback

The correct answer is: An augmenting path is found using ____________ on the residual graph → breadth first seach, forward edge → An edge in the residual graph that is also in the original graph,  ui,j - xi,j   → capacity of forward edge, xj,i → capacity of backward edge, backward edge → An edge in the the residual graph that is not in the original graph, residual graph → A flow graph derived from a preexisting flow graph and a flow on that graph
